# 키워드 검색량 분석을 위한 데이터 수집
## 목표 
- 크롤링을 통한 데이터 수집
- 수집한 데이터를 DB에 저장하고, 저장한 데이터 활용

## 사용 언어 및 라이브 러리
- 이 예제는 Python 3.9버전에서 활용하였으며, Pandas, Numpy, Matplotlib 라이브러리를 Django에서 구동

## 특징
- 데이터 사이언스 패키지 사용
- 수집한 데이터를 저장하고 활용할 수 있도록 DB 설계
